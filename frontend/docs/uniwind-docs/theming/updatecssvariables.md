## Résumé

Dynamically update CSS variables at runtime for specific themes

## Points importants

- runtime updates to CSS variables
- dynamic theme customization
- useful for user preferences, brand colors, and layout measurements

## Exemple / syntaxe typique

```tsx
Uniwind.updateCSSVariables('light', {
  '--color-primary': '#3b82f6',
})
```

## Attention

- Update variables that are known to Uniwind; define them statically when needed.
