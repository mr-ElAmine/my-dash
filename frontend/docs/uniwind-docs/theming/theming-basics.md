## Résumé

Set up themes in your React Native app with Uniwind. CSS-first theming with @theme, dark mode, custom colors, and dynamic theme switching — no ThemeProvider needed.

## Points importants

- default `light`, `dark`, and `system` themes
- theme variants like `dark:`
- `Uniwind.setTheme()` to switch themes
- `useUniwind()` to read runtime theme state

## Exemple / syntaxe typique

```tsx
Uniwind.setTheme('dark')
Uniwind.setTheme('system')
```

## Attention

- Use semantic tokens for colors instead of hardcoded values.
- Test all supported themes.
