## Résumé

React hook for accessing the current theme and reacting to theme changes

## Points importants

- read current theme
- track adaptive/system theme status
- rerender components when theme changes

## Exemple / syntaxe typique

```tsx
const { theme, hasAdaptiveThemes } = useUniwind()
```

## Attention

- Keep business logic theme-agnostic when possible; use variants for visual changes.
