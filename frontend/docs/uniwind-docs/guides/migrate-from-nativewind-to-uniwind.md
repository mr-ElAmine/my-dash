## Résumé

Drop-in migration from NativeWind to Uniwind in under 5 minutes. No Babel preset, full Tailwind v4, 2x faster — with a step-by-step guide and a one-shot AI skill.

## Points importants

- drop-in migration from NativeWind
- remove NativeWind-specific Babel/config where applicable
- configure Uniwind Metro wrapper
- keep Tailwind v4-compatible setup

## Exemple / syntaxe typique

```bash
npm remove nativewind
npm install uniwind tailwindcss
```

## Attention

- Validate custom NativeWind-specific APIs before assuming perfect equivalence.
- Clear Metro cache after migration.
