## Utilisation conseillée

- Donne ce dossier à Claude Code/Cursor comme documentation locale rapide.
- Utilise `sources.json` pour retrouver la page officielle correspondant à chaque fiche.
- Utilise `source/llms.txt` pour l’index officiel brut.

## Limite volontaire

Ces fichiers sont des fiches Markdown condensées et structurées. Ce n’est pas une copie brute intégrale mot-à-mot de toutes les pages.
